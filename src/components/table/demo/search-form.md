## zh-CN

设置 `column.search=true`, 结合 valueType，可以生成快速过滤项, 也可能通过表格 `search.items` 来自定义过滤项。

- 当使用本地数据时，需要手动处理数据的搜索。
- 当使用远程数据时，表格会把 `filter` 参数传递给 `request`。

> valueType 为函数类型时不支持。

## en-US

Set `column.search=true`, and combine with `valueType`, you can generate filtering form items, or you can customize filtering items through the `search.items` of the table.

- When using local data, you need to manually handle the search of the data.
- When using remote data, the table will pass the `filter` parameter to `request`.
